# DevCaps - Startup Ecosystem Platform

DevCaps is a full-stack web application built to connect startups, investors, and mentors in one place. We're making it easier for early-stage companies to find funding, guidance, and the resources they need to scale.

## Quick Start

1. Clone the repo and install dependencies in both `frontend` and `backend` folders
2. Set up your `.env` file with MongoDB connection and necessary configs
3. Run `npm start` in backend and `npm run dev` in frontend
4. Head to http://localhost:5173 and you're good to go

## Features

- Browse and showcase startups with detailed project portfolios
- Admin dashboard to manage projects, success stories, and newsletter subscribers
- Image cropping at 450x350 with automatic backend storage
- Newsletter subscription with email validation
- Success stories section to highlight customer wins

## Tech Stack

Frontend: React + Vite + Tailwind CSS | Backend: Node.js + Express + MongoDB
